# Blog API

API REST développée pour la gestion d’articles de blog.  
Projet réalisé dans le cadre du TAF 1 de l’UE INF222 (Développement Backend).

---

## Fonctionnalités

- Créer un article (titre, contenu, auteur, catégorie, tags)
- Lire tous les articles (avec filtres par catégorie, auteur, date)
- Lire un article spécifique (par ID)
- Modifier un article
- Supprimer un article
- Rechercher dans le titre ou le contenu



## Technologies utilisées

- Node.js – environnement d’exécution
- Express – framework web
- SQLite3– base de données légère
- CORS – gestion des requêtes cross-origin
- Poatman -testeur de serveur
